## Configuration File

Cicada uses `~/.cicada/config.yaml` for configuration.

```yaml
version: "1"

aws:
  profile: default
  region: us-west-2

sync:
  concurrency: 4
```

For complete current documentation, see the [README](https://github.com/scttfrdmn/cicada#readme).
