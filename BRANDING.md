**Last Updated:** 2025-11-25

Meet the Cicada mascot - a friendly, scholarly cicada representing the project's mission to help research labs organize and manage their scientific data. The mascot features:

- **Glasses**: Representing the scholarly, scientific focus of the platform
- **Backpack**: Ready for data collection and field work
- **Book**: Symbolizing documentation, knowledge, and data organization
- **Stripes & Wings**: Classic cicada appearance

---

## Project Name

**Official Name:** Cicada

**Tagline:** "A small lab data commons platform"

**Full Description:** Cicada is a lightweight data commons platform designed for small research laboratories. It helps labs organize research data, sync with cloud storage, extract metadata from scientific file formats, and preserve data provenance.

---

## Logo Usage

### Primary Logo

The mascot serves as the primary visual identity for Cicada.

**Available Formats:**
- PNG: `assets/images/cicada-mascot.png` (2.0 MB, high resolution)

**Usage Guidelines:**
- Minimum size: 200px width for digital, 1 inch for print
- Clear space: Maintain at least 50px clear space around the logo
- Do not modify, rotate, or distort the mascot
- Do not change colors or add effects
- Do not place on busy backgrounds that reduce visibility

### Acceptable Uses

✅ **Do:**
- Use in project documentation
- Use in presentations about Cicada
- Use in academic papers citing Cicada
- Use in blog posts or articles about the project
- Use in social media posts about Cicada
- Use as avatar/profile picture for official Cicada accounts

❌ **Don't:**
- Use for commercial purposes without permission
- Modify or create derivative works
- Use in a way that implies endorsement of other products
- Use in misleading or offensive contexts

## Typography

### Headings

**Font Family:** System sans-serif stack
```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
```

**Sizes:**
- H1: 2.5rem (40px) - Page titles
- H2: 2rem (32px) - Section headings
- H3: 1.5rem (24px) - Subsections
- H4: 1.25rem (20px) - Minor headings

### Body Text

**Font Family:** Same as headings
**Size:** 1rem (16px)
**Line Height:** 1.6

### Code

**Font Family:** Monospace stack
```css
font-family: "SFMono-Regular", Consolas, "Liberation Mono", Menlo, Courier, monospace;
```

## Voice and Tone

### Brand Personality

Cicada's communication style should be:

- **Approachable**: Friendly and welcoming, not intimidating
- **Clear**: Direct and easy to understand
- **Scientific**: Accurate and technically sound
- **Helpful**: Solution-oriented and supportive
- **Practical**: Focused on real-world research needs

### Writing Style

**Do:**
- Use clear, concise language
- Provide concrete examples
- Explain technical concepts when necessary
- Address users directly ("you", "your")
- Use active voice

**Don't:**
- Use unnecessary jargon
- Make assumptions about expertise
- Be condescending or overly casual
- Use marketing buzzwords
- Write overly long sentences

### Example Comparisons

❌ **Not this:**
> "Cicada revolutionizes data management with cutting-edge synergistic solutions for next-generation research paradigms."

✅ **This:**
> "Cicada helps research labs organize their data and sync it to cloud storage."

## Visual Elements

### Icons

When using icons alongside the mascot:
- Use simple, line-based icons
- Maintain consistent stroke width (2px)
- Use colors from the brand palette
- Align with Feather Icons or similar style

### Photography

When using photos in documentation or marketing:
- Show real research environments and equipment
- Include diverse researchers and lab settings
- Maintain natural lighting and authentic moments
- Avoid overly staged or stock-photo appearance

### Illustrations

Additional illustrations should:
- Match the mascot's friendly, approachable style
- Use the brand color palette
- Have rounded corners and smooth lines
- Include thoughtful details without being busy
